## 讲义领取分P
网站里有“网课分P设置”。
如果 B 站中间有一个“讲义领取”分P，就填写它的P号。系统会自动让后面的课时顺延。
例如：讲义领取是P6，则课时1-5对应P1-5，课时6对应P7。
