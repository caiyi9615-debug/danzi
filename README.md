# 电路原理每日复习+健身日程网站

按用户提供的每日复习+健身日程表制作：
- 固定作息模板
- 每日 6—7.5 小时复习
- 健身减肥时间
- 午睡和晚上休闲
- 电路原理邱关源1—8章
- 蜂考B站课时跳转
- 默认跳过P2讲义领取
- 云同步、打卡、导出、打印、一键清空

Render 环境变量：
SUPABASE_URL
SUPABASE_SERVICE_ROLE_KEY

Supabase 运行 supabase.sql。


## 本版显示修正
网课卡片不再显示“B站P几”，而是显示具体课时名称：
- 课时01 电路模型和电路定律
- 课时02 电阻电路的等效变换
- ...
- 课时11 相量法

跳转逻辑仍然自动跳过 P2 讲义领取。


## 本版新增

- 健身加入每日打卡项：`健身+爬坡有氧完成`
- 网课卡片不再写“参考时长”
- 部署到 Render 后，后端会通过 B站接口读取每个分P真实时长
- P2 讲义领取会自动跳过：课时01→P1，课时02→P3，课时11→P12

如果本地直接打开 HTML，由于没有后端 `/api/bili/pages`，真实时长会显示“正在获取真实时长”。
部署到 Render 后才会自动读取真实时间。
